# Frontend Tech Stack for Portfolio

## Core Technologies
1. **Next.js 14** - React framework with App Router
2. **TypeScript** - Type-safe JavaScript
3. **Tailwind CSS** - Utility-first CSS framework

## UI Component Libraries
4. **shadcn/ui** - High-quality React components
5. **Kibo UI** - Additional component library (optional)

## Animations & Styling
6. **Framer Motion** - Smooth animations

## Other Key Dependencies
- React 18
- React DOM 18
- ESLint for code quality
